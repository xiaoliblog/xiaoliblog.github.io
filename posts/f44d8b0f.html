Vue项目——电商管理系统之主页布局实现

总字数: 2.1k | 阅读时长: 11分钟

主页实现

主页布局

  • 使用ElementUI的布局容器组件，在配置文件中按需引入。注意出现-需要转为大写字母代替

token验证

  • 需要授权的 API ，必须在请求头对象中使用 Authorization 字段提供 token 令牌
  • 通过axios请求拦截器添加token，保证拥有获取数据的权限

  • 在main.js中配置

获取侧边栏数据

  • 使用生命周期钩子发送axios获取数据，赋值给data中的menulist，然后双层for循环遍历

实现折叠效果

  • 添加一个div做按钮效果，通过操作菜单的属性值collapse实现，是一个布尔值
  • 然后关闭折叠动画collapse-transition防止出现问题。宽度随collapse的布尔值变化

首页路由重定向

  • 实现登录到主页后重定向到/welcome，显示welcome组件的内容
  • 路由为home下的子路由，首先新建home组件
  • 配置路由规则
  • 然后在home组件的内容区域使用占位符

侧边栏由链接

  • 实现侧边栏点击不同选项实现不同页面展示
  • 使用菜单栏的属性router：是否使用 vue-router 的模式，启用该模式会在激活导航时以 index 作为 path 进行路由跳转
  • 我们使用响应数据里的path作为index
